# Sachin Egg Traders

A simple responsive website for Sachin Egg Traders, a wholesale egg supplier in Purulia, West Bengal.

## What’s inside

- `sachin_egg_traders.html` - main website page
- `styles.css` - site styles
- `scripts.js` - mobile navigation and reveal effects
- `assets/` - logo and favicon files

## Local preview

Open `sachin_egg_traders.html` in a browser, or run a simple local server:

```bash
python3 -m http.server 8000
```

Then visit:

```text
http://localhost:8000
```
