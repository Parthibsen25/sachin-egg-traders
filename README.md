# Sachin Egg Traders

A simple responsive website for Sachin Egg Traders, a wholesale egg supplier in Purulia, West Bengal.

## What’s inside

- `sachin_egg_traders.html` - main website page
- `styles.css` - site styles
- `scripts.js` - mobile navigation and reveal effects
- `assets/` - logo and favicon files

## Local preview

Open `sachin_egg_traders.html` in a browser, or run a simple local server:

```bash
python3 -m http.server 8000
```

Then visit:

```text
http://localhost:8000
```

## GitHub

Repository: `https://github.com/Parthibsen25/sachin-egg-traders.git`

## GitHub Pages

To publish this site with GitHub Pages:

1. Open the repository on GitHub.
2. Go to `Settings` > `Pages`.
3. Under `Build and deployment`, select `Deploy from a branch`.
4. Choose branch `main` and folder `/ (root)`.
5. Save the settings and wait for GitHub to publish the site.

After deployment, your site will usually be available at:

```text
https://parthibsen25.github.io/sachin-egg-traders/
```
